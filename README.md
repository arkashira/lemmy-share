# LemmyHub Search Functionality

This project implements the core search functionality for LemmyHub.

## Features

* Search bar is visible and accessible on the main page
* Users can input search queries and receive relevant results
* Results are filtered by relevance and include post title, content, and community

## Usage

1. Run the project using `python src/search.py`
2. Enter your search query when prompted
3. View the search results
